Practice 09 — Game Development with Pygame

This practice introduces game development using the Pygame library. Three mini-projects were implemented.

Projects

Moving Ball
A simple game where a red ball moves around the screen using arrow keys. The ball cannot leave the screen boundaries.

Music Player
An interactive music player controlled by keyboard. Supports play, stop, next and previous track. Place your audio files in the music folder.

Controls: P Play · S Stop · N Next · B Back · Q Quit

Mickey's Clock
A clock application featuring Mickey Mouse. The image of Mickey is displayed as the clock face, with his arms rotating as minute and second hands in real time.

Requirements

pip install pygame
